Developer Documentation

Project: School Management System (SaaS)
Core Framework: Laravel 12.x
Architecture: Modular Multi-tenant

1. Core Architecture

Modular System

Organized into 14 autonomous modules using nWidart/laravel-modules. Each module acts as a self-contained application layer.

Multi-Tenancy

Single-database approach. Tenant resolution via subdomain/custom-domain/API header. Data isolation enforced via school_id global scopes.

3. Authentication & Authorization

  • Auth: Session (Web) & Sanctum (API).
  • RBAC: Spatie permissions with tenant scoping logic.

4. Event System

Cross-module automation via central EventServiceProvider. Prevents hard dependencies between disparate modules.

5. Deployment

1. composer install
2. php artisan migrate
3. php artisan db:seed
4. php artisan storage:link
